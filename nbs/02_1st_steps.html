1st Steps#

Installation#

To keep it short we just show code snippets. Assumptions:

  • you have conda or minicondainstalled.

  • like to use jupyter lab as UI, otherwuse change code below accordingly

  • want to publish the book on github pages using ghp-import

create conda enviroment#

use the following code in the conda prompt

conda create -n jupyterbook python=3.10
conda activate jupyterbook
conda install -c conda-forge jupyter-book
conda install pandas
conda install -c conda-forge matplotlib
conda install -c conda-forge jupyterlab
conda install -c conda-forge ghp-import

“git-stuff”#

  • maybe explain create repo, github pages and github actions

1st steps create book#

  • https://jupyterbook.org/en/stable/start/create.html

create book template and test setup#

Create a template book using the following code in the conda prompt

jupyter-book create mybook/

Test setup (don’t move to directory mybook now)

jupyter-book build mybook/

check index.html in _build/html/

housekeeping#

because jb build mybook/ created a template lets replace the defaults w.r.t. our needs

  1. create folder for content files (notebooks, markdown,…), e.g. ǹbs

  2. move intro.md to folder nbs

  3. adjust _toc.yml such that header looks like

# Table of contents
# Learn more at https://jupyterbook.org/customize/toc.html

format: jb-book
root: nbs/intro

  1. remove unessary template files

Start page intro.md#

This file will become the starting page of your book,, i.e. index.html in case of a web-book.

Hence add some content

1st content page#

simply create a first content page, e.g. 01_content.ipynb

Table of content _toc.yml#

_toc.yml tells which files and in which order files become part of the book, e.g.:

# Table of contents
# Learn more at https://jupyterbook.org/customize/toc.html

format: jb-book
root: nbs/intro
chapters:
- file: nbs/01_content

or more information see TOC-Doc.

_config.yml#

define book settings, for more see Configuration reference

  1. adjust title

  2. name author

  3. (optional) remove full execution on build (execute: and execute_notebooks: force)

  4. set repository location

example#
# Book settings
# Learn more at https://jupyterbook.org/customize/config.html

title: A slim introduction to jupyter books
author: Christian Barz
logo: logo.png

# Define the name of the latex output file for PDF builds
latex:
  latex_documents:
    targetname: book.tex

# Add a bibtex file so that we can create citations
bibtex_bibfiles:
  - references.bib

# Information about where the book exists on the web
repository:
  url: https://github.com/barz-christian/jupyterbook  # Online location of your book
  path_to_book: docs  # Optional path to your book, relative to the repository root
  branch: master  # Which branch of the repository should be used when creating links (optional)
